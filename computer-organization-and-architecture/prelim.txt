Day 1

Which level in the multilevel machine model deals directly with transistors and logic gates?
A. L0
B. L1
C. L2
D. L4
Answer: A

Which of the following best describes machine language?
A. A symbolic language for humans to read
B. The primitive instruction set directly executed by hardware
C. A set of high-level abstractions used to bridge the gap between hardware and software
D. A collection of OS commands
Answer: B

Who proposed the concept of microprogramming in 1951?
A. Alan Turing
B. Maurice Wilkes
C. John von Neumann
D. Donald Knuth
Answer: B

The assembler’s primary role is to:
A. Interpret machine code to turn it into electrical signals
B. Convert assembly language to machine-level code
C. Compile high-level code into bytecode
D. Manage cache memory
Answer: B

Which early computing practice required programmers to personally operate the computer and manually load programs?
A. Batch processing
B. Open shop programming
C. Timesharing
D. Client-server computing
Answer: B

In the 1960s, operating systems were introduced primarily to:
A. Replace machine language with high-level languages
B. Reduce hardware costs
C. Simplify and speed up program execution
D. Replace interpreters with compilers
Answer: C

Why did some architectures eventually remove microprogramming?
A. It was too expensive to store the microcode
B. It slowed down execution compared to direct hardware execution
C. It made compilers unnecessary
D. It increased instruction set size which added lots of complexity
Answer: B

Elimination of Microprogramming – Why did some architectures later remove microprogramming? What were the trade-offs?

A. Store large amounts of data
B. Perform arithmetic and logic operations
C. Fetch and decode instructions from memory
D. Connect the CPU to external devices
Answer: C

Which register holds the address of the next instruction to be executed?
A. IR
B. PC
C. MAR
D. MDR
Answer: B

In Von Neumann architecture, which of the following is NOT a typical component of the data path cycle?
A. ALU
B. Registers
C. Prefetch Buffer
D. GPU
Answer: D

The fetch-decode-execute cycle begins with:
A. Executing the instruction
B. Fetching the instruction from memory
C. Incrementing the program counter
D. Loading data from registers
Answer: B

What is an interpreter in the context of CPU design?
A. A device that converts analog signals to digital
B. A hardware unit that translates high-level code into binary
C. A program that imitates the function of a CPU
D. A part of the operating system that handles I/O
Answer: C

Which of the following was a design goal of RISC computers?
A. Maintain backward compatibility with older architectures
B. Minimize instruction count but maximize execution speed
C. Emulate CISC instructions for compatibility
D. Increase semantic gap between hardware and high-level languages

Pipelining improves performance by:
A. Increasing clock speed
B. Executing multiple instructions in different stages simultaneously
C. Reducing register usage
D. Allowing the CPU to skip decoding steps

In SIMD (Single Instruction Multiple Data),
A. Each core executes different instructions on the same data
B. One instruction is applied to multiple data items in parallel
C. Multiple programs run on a single processor
D. Each processor has its own control unit

Which is loosely coupled:
A. Multiprocessors
B. Multicomputers
